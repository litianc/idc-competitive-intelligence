# 周报格式规范文档

## 格式类型

本系统实现**简洁版Markdown周报**，适合日常快速浏览和信息获取。

## 中文商业格式规范

### 1. 编号系统

**一级标题编号**：使用中文数字
```markdown
## 一、投资动态
## 二、技术进展
## 三、政策法规
## 四、市场动态
```

**列表项编号**：使用阿拉伯数字
```markdown
1. 第一条新闻
2. 第二条新闻
```

### 2. 标签格式

使用全角方括号【】标识分类标签：
```markdown
【投资】
【技术】
【政策】
【市场】
【高优先级】
【中优先级】
【低优先级】
```

**错误示例**：
- ❌ [投资] - 半角括号
- ❌ 「投资」 - 日文引号
- ❌ (投资) - 圆括号

### 3. 标点符号

使用中文全角标点符号：
```
、 顿号（并列）
， 逗号
。 句号
； 分号
： 冒号
！ 感叹号
？ 问号
【】 方括号
《》 书名号
" " 双引号
' ' 单引号
```

**例外**：英文内容中保持英文标点
```markdown
NVIDIA发布H200 GPU，性能提升40%
```

### 4. 日期格式

统一使用ISO格式：
```
2025-01-03
2025-11-23
```

**不使用**：
- ❌ 2025/01/03
- ❌ 2025.01.03
- ❌ 01-03-2025

### 5. 元数据行格式

来源和日期在同一行，使用竖线分隔，前后有空格：
```markdown
【投资】中国IDC圈 | 2025-01-03
【技术】数据中心世界 | 2025-01-02
```

**格式要求**：
- 分类标签在最前
- 来源媒体名称
- 空格 + 竖线 + 空格
- 日期
- 整行不超过80字符

## 格式规范细则

### 1. 标题层级

```markdown
# 一级标题（周报标题）
## 二级标题（模块标题）
### 三级标题（可选，用于子模块）
```

### 2. 文章条目结构

每条新闻包含4个元素：

```markdown
序号. 文章标题
   元数据行：【分类】来源 | 日期
   摘要内容（80-150字，缩进3个空格）
   链接行：[详情](URL)

空行分隔下一条
```

**格式要求**：
1. 标题行：序号 + 点 + 空格 + 标题
2. 元数据行：缩进3个空格
3. 摘要内容：缩进3个空格，可多行
4. 链接行：缩进3个空格
5. 条目间用空行分隔

### 3. 分隔线

使用三个连字符加空行：
```markdown
---

## 下一个模块
```

### 4. 引用块

周报元信息使用引用块：
```markdown
> 报告周期：2025-01-01 至 2025-01-07
> 生成时间：2025-01-07 17:00
> 本周共收录 XX 条行业动态
```

### 5. 模块标题格式

```markdown
## 一、投资动态【高优先级】
## 二、技术进展【高优先级】
## 三、政策法规【高优先级】
## 四、市场动态【高优先级】
## 五、其他动态【中低优先级】
```

**规则**：
- 使用二级标题（##）
- 中文数字 + 顿号 + 模块名称 + 优先级标签
- 如果某模块无内容，可省略该模块

### 6. 文件命名规范

```
IDC行业周报_2025年第1周.md
IDC行业周报_2025年第2周.md
IDC行业日报_2025-01-03.md
```

**规则**：
- 周报：`IDC行业周报_YYYY年第X周.md`
- 日报：`IDC行业日报_YYYY-MM-DD.md`

## 内容质量标准

### 1. 摘要要求

- **长度**：80-150字（中文字符）
- **结构**：核心事实 + 关键数据 + 影响分析
- **语言**：专业、简洁、客观
- **避免**：主观评论、过度修饰、重复标题

### 2. 标题要求

- **长度**：10-30字
- **清晰**：准确概括文章主题
- **避免**：标题党、过度简化

### 3. 链接要求

- 必须是有效链接（link_valid=true）
- 使用原文链接，不使用短链接
- 链接文本统一为"详情"

### 4. 模块完整性

- 每个模块至少包含1条高优先级新闻
- 如果某模块无高优先级新闻，可归入"其他动态"
- "其他动态"可包含所有中低优先级新闻，摘要可简化

```markdown
   [详情](https://www.idcquan.com/article/123456)
```

### ❌ 错误示例

```markdown
1. 某公司完成10亿元融资用于AI算力中心建设
[投资] 中国IDC圈 2025/01/03  ← 错误：半角括号、日期格式错误
该公司完成了新一轮融资!!!  ← 错误：无实质内容、多余感叹号
https://www.idcquan.com/article/123456  ← 错误：直接URL，无[详情]标签
```

## 自动化生成注意事项

1. **优先级排序**：同一模块内，高分文章在前
2. **时间排序**：同等优先级按发布日期倒序
3. **去重**：同一文章只出现一次
4. **模块平衡**：确保4个主要模块都有内容
5. **链接验证**：生成前验证所有链接有效性

- [ ] 元数据行格式：【分类】来源 | 日期
- [ ] 每条新闻包含标题、元数据、摘要、链接
- [ ] 摘要长度80-150字
- [ ] 所有链接可访问
- [ ] 至少包含4个主要模块
- [ ] 文件命名符合规范
