# 中国IDC行业媒体源研究文档

## 研究目标

为竞争情报系统选择3-5个中国IDC行业核心媒体源，确保：
1. 覆盖投资、技术、政策、市场四大类内容
2. 信息权威可信
3. 更新及时
4. 技术上可采集

## 日期提取策略

### 标准格式

```python
# YYYY-MM-DD
"2025-01-03"

# YYYY年MM月DD日
"2025年1月3日"
```

### 相对时间

```python
# X天前 → 计算实际日期
"3天前" → date.today() - timedelta(days=3)

# X小时前 → 计算实际日期
"5小时前" → date.today()  # 当天

# 昨天/今天
"昨天" → date.today() - timedelta(days=1)
```

### 不完整日期

```python
# MM-DD（补充当年）
"01-03" → f"{datetime.now().year}-01-03"
```

## 数据质量标准

### 必需字段

- ✅ 标题
- ✅ 原文链接（必须可访问）
- ✅ 来源媒体
- ✅ 发布日期（实际发布日期，非采集时间）
- ✅ 内容正文

### 可选字段

- 作者
- 摘要（如网站提供）
- 标签/分类
- 阅读量
- 评论数

### 质量检查

1. **链接有效性**：HTTP状态码200
2. **日期合理性**：不能是未来日期
3. **内容完整性**：正文至少100字
4. **去重**：基于URL hash去重

## 法律和道德考虑

### 合规原则

1. **Robots.txt遵守**：检查并遵守网站robots.txt
2. **请求频率控制**：避免对网站造成负担
3. **User-Agent标识**：使用真实的User-Agent
4. **仅用于研究**：数据仅用于内部情报分析

### 使用限制

- ❌ 不可商业转载
- ❌ 不可去除原文链接
- ✅ 仅内部使用
- ✅ 保留来源标注
